Configurar Supabase

Conecta el sistema a tu base de datos antes de comenzar. Solo necesitas hacerlo una vez.

1
Crea un proyecto en supabase.com Es gratis. Espera ~2 minutos a que esté listo.
2
Obtén tus credenciales En tu proyecto: Settings → API → copia Project URL y anon public key.
3
Ejecuta el SQL de inicialización En Supabase ve a SQL Editor y ejecuta el script que está al final de este archivo HTML.
